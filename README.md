# MyWrapper Technologies — Console

AI 客服 Agent 管理平台前端 v2.0

## 本地開發

```bash
cp .env.local.example .env.local
# 設定 NEXT_PUBLIC_API_URL=http://localhost:4000

npm install
npm run dev
```

開啟 `http://localhost:3000`

## Zeabur 部署

連接此 GitHub repo，設定以下環境變數：
```
NEXT_PUBLIC_API_URL=https://your-backend.zeabur.app
```
